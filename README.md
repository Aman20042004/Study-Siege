# 📚 Study Siege

Study Siege is a modern Android productivity application designed to help students build discipline, improve focus, and track their study performance through meaningful analytics.

## ✨ Features

- Smart Task Management
- Relevant, Gap & Not Core task categorization
- Stopwatch & Timer modes
- Custom study timer
- Pause & Resume study sessions
- Break tracking and statistics
- Daily productivity analytics
- Interactive study calendar
- Focus Score
- Consistency Streak
- Most Wasted Time analysis
- Top Tasks insights
- Offline-first architecture using Room Database

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Room Database
- Kotlin Coroutines
- StateFlow
- Navigation Compose
- Gradle Kotlin DSL
- Version Catalogs

## 🚧 Project Status

Study Siege is currently under active development.

Upcoming features include:

- Secure Authentication
- Cloud Synchronization
- Multi-device Support
- Backend Integration
- Enhanced Analytics
- Achievement System
